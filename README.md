# yazi-config

My yazi file manager configuration with nightfly flavor.

## What's included

- `config.toml` — editor and flavor settings
- `package.toml` — plugin and flavor dependencies
- `theme.toml` — theme flavor selection
- `flavors/` — tokyo-night and nightfly color schemes

## Install

```bash
./install.sh
```

## Symlinks

| Source | Target |
|--------|--------|
| `config.toml` | `~/.config/yazi/config.toml` |
| `package.toml` | `~/.config/yazi/package.toml` |
| `theme.toml` | `~/.config/yazi/theme.toml` |
| `flavors/` | `~/.config/yazi/flavors` |
